4. gunicron --bind 0.0.0.0:7000 ping:app

5. Or we can combine 3 & 4 steps: pipenv run gunicron --bind 0.0.0.0:7000 ping:app

6. docker build -t test .
7. docker run -it --rm --entrypoint=bash test
